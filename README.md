# Sistema de Control y Monitoreo de Ansiedad y Estrés Estudiantil

## Resumen del Proyecto

Esta aplicación web está diseñada para monitorear y controlar los niveles de estrés y ansiedad en estudiantes de instituciones educativas. La plataforma surge de la necesidad de detectar y atender casos de estudiantes que presentan niveles alarmantes de estrés y ansiedad, permitiendo a las instituciones educativas proporcionar herramientas de apoyo adecuadas e identificar a los estudiantes más afectados.

## Contexto del Problema

Se ha identificado un incremento preocupante en los reportes de estudiantes que presentan niveles críticos de estrés y ansiedad en el ámbito educativo. Estos problemas de salud mental pueden impactar significativamente en el rendimiento académico y bienestar general de los estudiantes. 

La aplicación busca:
- **Detectar tempranamente** estudiantes con niveles altos de estrés y ansiedad
- **Proporcionar herramientas** de evaluación y seguimiento
- **Segregar información** por institución educativa para un mejor control
- **Facilitar el acceso** a tests especializados para evaluación de niveles
- **Generar reportes** para la toma de decisiones institucionales

## Tecnologías Utilizadas

- **HTML5**: Estructura y contenido de las páginas web
- **CSS3**: Estilos, diseño responsivo y experiencia de usuario
- **JavaScript**: Interactividad, validaciones y funcionalidad dinámica del lado cliente
- **PHP**: Lógica del servidor, procesamiento de datos y conexión con base de datos

## Características Principales

### 🎯 Evaluación de Niveles
- Tests estandarizados para medir niveles de estrés
- Cuestionarios especializados para evaluación de ansiedad
- Sistema de puntuación y clasificación de resultados

### 📊 Panel de Control Institucional
- Dashboard para visualización de estadísticas por escuela
- Reportes detallados de estudiantes en riesgo
- Análisis de tendencias y patrones

### 👥 Gestión de Usuarios
- Sistema de autenticación para estudiantes y administradores
- Perfiles diferenciados por tipo de usuario
- Acceso segregado por institución educativa

### 📈 Monitoreo y Seguimiento
- Historial de evaluaciones por estudiante
- Alertas automáticas para casos críticos
- Sistema de notificaciones para administradores

## Instalación y Configuración

### Requisitos del Sistema
- Servidor web (Apache/Nginx)
- PHP 7.4 o superior
- MySQL/MariaDB
- Navegador web moderno

### Pasos de Instalación
1. Clonar el repositorio
2. Configurar la base de datos
3. Establecer permisos de archivos
4. Configurar variables de entorno
5. Ejecutar scripts de inicialización

## Uso del Sistema

### Para Estudiantes
1. Registro en la plataforma con credenciales institucionales
2. Acceso a tests de evaluación disponibles
3. Realización de cuestionarios de estrés y ansiedad
4. Visualización de resultados personales

### Para Administradores
1. Acceso al panel de control institucional
2. Revisión de reportes y estadísticas
3. Identificación de estudiantes en riesgo
4. Generación de informes ejecutivos
